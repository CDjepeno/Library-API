## Features
- Test fonctionnel avec Jest.
- Utilisation de swagger docs pour documenter l'api
- MongoDB
- Authentification JWT Token
- CRUD

## Getting Started
```bash
$ git clone https://github.com/CDjepeno/Library-API.git
$ cd Hotel_API-Node-js
$ npm install
$ npm start
$ http://localhost:5000
```
